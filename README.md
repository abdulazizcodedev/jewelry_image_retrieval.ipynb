# jewelry_image_retrieval.ipynb

## Jewelry Image Retrieval Project

This project trains deep learning models (VGG16, ResNet50)
and performs image retrieval using cosine similarity.

## How to Run
1. Open the notebook in Google Colab
2. Enable GPU (Runtime → Change runtime type → GPU)
3. Run all cells

## Dataset
The dataset is automatically downloaded from Hugging Face:
https://huggingface.co/datasets/sidd707/jewelry-design-dataset
